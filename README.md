# CREA8 Academy Website

एक responsive, single-page institute website. कोई build step या framework आवश्यक नहीं है।

## Publish करने से पहले

1. `index.html` में **CREA8 Academy**, फोन, ईमेल और address को अपनी details से बदलें।
2. सभी `919876543210` WhatsApp numbers को अपना country-code सहित नंबर दें।
3. `instagram.com/crea8academy` और `facebook.com/crea8academy` को अपनी वास्तविक profile links से बदलें।
4. Placement cards में demo names, companies और packages को वास्तविक data से बदलें।

## Hosting

पूरे `creative-edge-institute` folder को hosting के `public_html` folder में upload करें। Netlify पर folder को drag-and-drop भी कर सकते हैं। Homepage file `index.html` है।

## Files

- `index.html` — content और page structure
- `styles.css` — design और responsive layout
- `script.js` — menu, animations, filters और WhatsApp enquiry
- `assets/hero-students.png` — generated hero banner
