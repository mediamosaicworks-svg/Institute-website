GUIDE

How to Use the Admin Panel

  1. Select Website Details, Page Text, Banners, Portfolio or Placements from the left.
  2. Edit the information or upload an image.
  3. Click the green Save button.
  4. Click Download for GitHub, then replace the repository's site-data.js with the downloaded file.
